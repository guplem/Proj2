 #### # Sorting layers (sprite)
  - [-100] --> Ambient Light
  - [0, 50] --> Background world elements
  - [51, 60] --> Floor
  - [61, 75] --> Between player and bg (like bulbs)
  - [100] --> Player crouched
  - [101, 149] --> Hiding objects
  - [150] --> Player (standard)
  - [151-200] --> Enemies/NPC
  - [201-300] --> Decorative elements
